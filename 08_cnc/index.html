Design Something and Make It Using CNC</h4> For this part of the assignment, I wanted to create a birthday present for my friend, a jewelry tray. I started by designing the model in Fusion 360. <p class = "margin"></p> <img src="./tray_fusion.png" alt="generic placeholder image" width = "400" height = "400"> <p class = "margin"></p> I then used ASPIRE and ShopBot to cut the design on the CNC machine. <p class = "margin"></p> <img src="./tray_shopbot.jpg" alt="generic placeholder image" width = "400" height = "400"> <p class = "margin"></p> This is how the initial piece came out: <p class = "margin"></p> <img src="./rough_tray.jpg" alt="generic placeholder image" width = "400" height = "400"> <p class = "margin"></p> After sanding it down, it looked much smoother: <p class = "margin"></p> <img src="./sanded_tray.jpg" alt="generic placeholder image" width = "400" height = "400"> <p class = "margin"></p> Finally, I applied a shellac finish to give it a polished look. Here’s the final result. I hope my friend likes it. <p class = "margin"></p> <img src="./polished_tray.jpg" alt="generic placeholder image" width = "400" height = "400"> <p class = "margin"></p> <h4>Second Iteration: Harvard “H” Jewelry Tray</h4> After gaining some experience with the ShopBot, I wanted to make something more complex, an irregular-shaped jewelry tray featuring the Harvard “H” logo in the center. Here’s what the 3D model looked like in Fusion 360: <p class = "margin"></p> <img src="./H_try_fusion.png" alt="generic placeholder image" width = "400" height = "400"> <p class = "margin"></p> I had faced issues with the Join command in Aspire while working on my previous model. This time, with Bobby’s guidance, I created a full 3D model and projected it onto a plane to get a single connected sketch. I used pocket cuts so that the middle “H” would sit lower than the surrounding edges. Here’s how it looked after cutting it on the ShopBot: <p class = "margin"></p> <img src="./unsanded_H_tray.jpg" alt="generic placeholder image" width = "400" height = "400"> <p class = "margin"></p> One section of the wood turned out to be hollow, so I filled it with wood filler and later reinforced it with wood glue. <p class = "margin"></p> <img src="./hole1_H_tray.jpg" alt="generic placeholder image" width = "400" height = "400"> <p class = "margin"></p> <p class = "margin"></p> <img src="./hole2_H_tray.jpg" alt="generic placeholder image" width = "400" height = "400"> <p class = "margin"></p> <h4>2. Post-Processing the Design</h4> Once the tray was fully sanded, I used vacuum forming to create a mold. <p class = "margin"></p> <img src="./vacuum_forming.jpg" alt="generic placeholder image" width = "400" height = "400"> <p class = "margin"></p> This is how the mold turned out: <p class = "margin"></p> <img src="./vacuum_mold.jpg" alt="generic placeholder image" width = "400" height = "400"> <p class = "margin"></p> Inspired by Kassia’s swirly paint designs on the state plates, I decided to experiment with swirled color effects for my plaster casting. After mixing the plaster powder and water in the correct ratio, I added a few drops of blue paint to achieve a marbled look. <p class = "margin"></p> <img src="./paint_swirl.jpg" alt="generic placeholder image" width = "400" height = "400"> <p class = "margin"></p> Then, I poured the mixture into the mold: <p class = "margin"></p> <img src="./H_cast.jpg" alt="generic placeholder image" width = "400" height = "400"> <p class = "margin"></p> After removing it from the mold, I was really happy with how it turned out. The swirls gave it a beautiful finish.
